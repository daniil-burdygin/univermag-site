Privacy Policy

Last updated: March 10, 2026

Univermag ("the Add-on") is a Google Sheets™ add-on developed by Daniil Burdygin. This privacy policy describes how the Add-on handles user data.

Data Collection

The Add-on does not collect, store, or transmit personal data to any external servers. All processing happens within the user's Google Apps Script environment.

Data Stored Locally

The following data is stored in Google Apps Script UserProperties (accessible only to the individual user):

  • Wildberries API tokens — provided by the user for API access. Stored securely in UserProperties and never shared.
  • Account names — user-defined labels for their Wildberries accounts.
  • Schedule configuration — auto-export settings (frequency, time).
  • Report history — last 6 report execution results (report name, row count, timestamp).

Data from Wildberries API

The Add-on fetches analytics data from the Wildberries API using tokens provided by the user. This data is written directly into the user's Google Sheets™ spreadsheet. The Add-on does not store, cache, or forward this data to any third party.

Google User Data

The Add-on requests the following Google OAuth scopes:

  • spreadsheets.currentonly — read/write access to the current spreadsheet only
  • script.external_request — to make HTTP requests to the Wildberries API
  • script.scriptapp — to create time-based triggers for scheduled exports
  • userinfo.email — to identify the user for email notifications

The Add-on does not access any other Google services or data beyond the scopes listed above.

Analytics

The Add-on sends anonymous usage events (report type, row count, NPS score) to Google Analytics via the Measurement Protocol. No personal information, API tokens, or spreadsheet content is included in these events.

Third-Party Services

The Add-on communicates only with:

  • Wildberries API (seller-analytics-api.wildberries.ru, statistics-api.wildberries.ru, content-api.wildberries.ru, marketplace-api.wildberries.ru, advert-api.wildberries.ru, feedbacks-api.wildberries.ru, common-api.wildberries.ru) — to fetch seller analytics data
  • Google Analytics (google-analytics.com) — anonymous usage tracking

Data Deletion

Users can delete all stored data at any time by removing their API tokens from the Add-on settings. Uninstalling the Add-on removes all UserProperties data.

Children's Privacy

The Add-on is not directed at children under 13 and does not knowingly collect data from children.

Changes to This Policy

This policy may be updated from time to time. The "Last updated" date at the top will reflect the most recent revision.
