Политика конфиденциальностиPrivacy Policy

Последнее обновление: 10 марта 2026 Last updated: March 10, 2026

Univermag («Расширение») — это расширение Google Sheets™, разработанное Daniil Burdygin. Настоящая политика конфиденциальности описывает, как Расширение обрабатывает данные пользователей. Univermag (“the Add-on”) is a Google Sheets™ add-on developed by Daniil Burdygin. This privacy policy describes how the Add-on handles user data.

Сбор данныхData Collection

Расширение не собирает, не хранит и не передаёт персональные данные на внешние серверы. Вся обработка происходит в среде Google Apps Script пользователя. The Add-on does not collect, store, or transmit personal data to any external servers. All processing happens within the user's Google Apps Script environment.

Данные, хранящиеся локальноData Stored Locally

Следующие данные хранятся в Google Apps Script UserProperties (доступны только конкретному пользователю): The following data is stored in Google Apps Script UserProperties (accessible only to the individual user):

  • API-токены WildberriesWildberries API tokens — предоставленные пользователем для доступа к API. Хранятся в UserProperties и никогда не передаются третьим лицам.provided by the user for API access. Stored securely in UserProperties and never shared.
  • Названия аккаунтовAccount names — пользовательские метки для аккаунтов Wildberries.user-defined labels for their Wildberries accounts.
  • Настройки расписанияSchedule configuration — параметры автовыгрузки (частота, время).auto-export settings (frequency, time).
  • История отчётовReport history — последние 6 результатов выгрузки (название, количество строк, время).last 6 report execution results (report name, row count, timestamp).

Данные из Wildberries APIData from Wildberries API

Расширение получает аналитические данные из Wildberries API с помощью токенов, предоставленных пользователем. Эти данные записываются непосредственно в таблицу Google Sheets™ пользователя. Расширение не хранит, не кэширует и не пересылает эти данные третьим лицам. The Add-on fetches analytics data from the Wildberries API using tokens provided by the user. This data is written directly into the user's Google Sheets™ spreadsheet. The Add-on does not store, cache, or forward this data to any third party.

Данные Google-аккаунтаGoogle User Data

Расширение запрашивает следующие OAuth-разрешения Google: The Add-on requests the following Google OAuth scopes:

  • spreadsheets.currentonly — чтение/запись только в текущей таблицеread/write access to the current spreadsheet only
  • script.external_request — HTTP-запросы к Wildberries APIto make HTTP requests to the Wildberries API
  • script.scriptapp — создание триггеров для автовыгрузки по расписаниюto create time-based triggers for scheduled exports
  • userinfo.email — идентификация пользователя для email-уведомленийto identify the user for email notifications

Расширение не обращается к другим сервисам или данным Google, кроме перечисленных выше. The Add-on does not access any other Google services or data beyond the scopes listed above.

АналитикаAnalytics

Расширение отправляет анонимные события использования (тип отчёта, количество строк, NPS-оценка) в Google Analytics через Measurement Protocol. Персональные данные, API-токены и содержимое таблицы не включаются в эти события. The Add-on sends anonymous usage events (report type, row count, NPS score) to Google Analytics via the Measurement Protocol. No personal information, API tokens, or spreadsheet content is included in these events.

Сторонние сервисыThird-Party Services

Расширение взаимодействует только с: The Add-on communicates only with:

  • Wildberries API (seller-analytics-api.wildberries.ru, statistics-api.wildberries.ru, content-api.wildberries.ru, marketplace-api.wildberries.ru, advert-api.wildberries.ru, feedbacks-api.wildberries.ru, common-api.wildberries.ru) — получение аналитических данныхto fetch seller analytics data
  • Google Analytics (google-analytics.com) — анонимная аналитика использованияanonymous usage tracking

Удаление данныхData Deletion

Пользователь может удалить все сохранённые данные в любой момент, удалив свои API-токены в настройках расширения. При удалении расширения все данные UserProperties удаляются автоматически. Users can delete all stored data at any time by removing their API tokens from the Add-on settings. Uninstalling the Add-on removes all UserProperties data.

Конфиденциальность детейChildren's Privacy

Расширение не предназначено для детей младше 13 лет и не собирает данные о детях. The Add-on is not directed at children under 13 and does not knowingly collect data from children.
