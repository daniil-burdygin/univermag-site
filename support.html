Часто задаваемые вопросыFAQ

Как получить API-токен Wildberries? How do I get a Wildberries API token?

Перейдите на seller.wildberries.ru/api-integrations, создайте новый API-ключ и выберите нужные категории токена (Аналитика, Статистика, Контент, Маркетплейс) в зависимости от того, какие отчёты вам нужны. Go to seller.wildberries.ru/api-integrations, create a new API key, and select the required token categories (Analytics, Statistics, Content, Marketplace) depending on which reports you need.

Какие категории токена нужны? Which token categories do I need?
  • АналитикаAnalytics — Воронка продаж, Платное хранение, Остатки WB, Продажи по регионамSales Funnel, Paid Storage, WB Warehouse Remains, Region Sales
  • СтатистикаStatistics — Заказы, Продажи, Остатки, Финансовый отчётOrders, Sales, Stocks, Financial Report
  • КонтентContent — Карточки товаровProduct Cards
  • МаркетплейсMarketplace — СкладыWarehouses
  • ПродвижениеPromotion — Рекламные кампанииAd Campaigns
  • ОтзывыFeedbacks — ОтзывыReviews
Отчёт прервался. Что делать? The report stopped before finishing. What do I do?

Google Apps Script имеет лимит выполнения 6 минут. Для больших наборов данных расширение сохраняет прогресс автоматически. Вы увидите баннер «Продолжить» — нажмите его, чтобы продолжить с места остановки. Google Apps Script has a 6-minute execution limit. For large datasets, the add-on saves progress automatically. You will see a "Resume" banner — click it to continue from where it stopped.

Можно ли использовать несколько аккаунтов? Can I use multiple Wildberries accounts?

Да. Добавьте несколько токенов в настройках. Переключайтесь между аккаунтами через выпадающий список в верхней части панели. Также можно использовать «Пакетный режим» для выгрузки отчёта по всем аккаунтам сразу. Yes. Add multiple tokens in the Settings tab. Switch between accounts using the dropdown at the top of the sidebar. You can also use "Batch mode" to run a report for all accounts at once.
