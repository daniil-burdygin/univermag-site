Frequently Asked Questions

How do I get a Wildberries API token?

Go to seller.wildberries.ru/api-integrations, create a new API key, and select the required token categories (Analytics, Statistics, Content, Marketplace) depending on which reports you need.

Which token categories do I need?

  • Analytics — Sales Funnel, Paid Storage, WB Warehouse Remains, Region Sales
  • Statistics — Orders, Sales, Stocks, Financial Report
  • Content — Product Cards
  • Marketplace — Warehouses
  • Promotion — Ad Campaigns
  • Feedbacks — Reviews

The report stopped before finishing. What do I do?

Google Apps Script has a 6-minute execution limit. For large datasets, the Add-on saves progress automatically. You will see a "Resume" banner — click it to continue from where it stopped.

Can I use multiple Wildberries accounts?

Yes. Add multiple tokens in the Settings tab. Switch between accounts using the dropdown at the top of the sidebar. You can also use "Batch mode" to run a report for all accounts at once.
