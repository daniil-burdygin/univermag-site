Univermag

Google Sheets™ Add-on for Wildberries sellers

Univermag is a Google Sheets™ add-on that helps Wildberries marketplace sellers export analytics data directly into spreadsheets. It connects to the Wildberries API and fetches reports with a single click.

Features

  • 13 report types: sales funnel, orders, sales, product cards, stocks, paid storage, financial reports, and more
  • Multi-account support with easy token switching
  • Scheduled auto-exports (daily, weekly, monthly)
  • Queue mode for batch report execution
  • Resume interrupted downloads automatically
  • All data stays in your Google Sheets™ — no external servers
